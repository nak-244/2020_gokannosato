五感を刺激・心の癒やし

五感の里では「聞く・見る・食べる・触る・嗅ぐ」の五感を大切にした介護サービスを提供しています。

人は、視覚・聴覚・触覚・味覚・嗅覚、いわゆる「五感が刺激」されることで、「幸せ感」が促されます。

また、在りし日の「楽しい思い出」を五感を通して呼び起こすことで、本来持っている人としての喜びを感じることができます。

それには、人と人との結びつきが大切で、共に喜びを分かち合える環境が必要です。

私達、「五感の里」では、人、物との触れ合いから五感を刺激し、脳活性化に向けた取り組みを行い、一方一方の人生に寄り添い、

幸せ、幸福感を味わって頂けるよう環境を整え、お越しをお待ち致しております。
